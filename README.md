# Scratch_App
This is a ReactJS-based visual programming environment inspired by MIT Scratch. It allows users to create, drag, and run code blocks for sprite animation and interactionâ€”making it educational and engaging for beginner coders.

->Drag & Drop Blocks: Users can drag motion and looks blocks to build logic.
->Sprite Animations:

->Collision Detection: Sprites detect collision and exchange actions dynamically. (Hero Feature)
->Custom UI: Built with ReactJS, Tailwind CSS, MUI icons, and react-beautiful-dnd.
